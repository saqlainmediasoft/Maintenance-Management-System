Release notes for Kendo UI v2013.1.514

Framework

What's new

  • Expose kendo.fx to TypeScript
  • Hidden PanelBar items in IE10 have additional spacing and a bullet when shown

What's fixed

  • Fx support for surface configuration option is broken
  • page turn effect is broken
  • TypeScript definitions do not include kendo.ns
  • Unbind doesn't detach the event handler if bound multiple times to the same event

DataSource

What's fixed

  • Pristine collection is not correctly updated when server returns empty object
  • Filtering string column that contains null values with "starts with" or "ends with" operator throws JS error
  • RequestStart and RequestEnd events are not raised during remove virtualization requests
  • Data is re-requested if all items are deleted and query is called with server operations disabled
  • Grouping on string or boolean and null does not yield correct result

Globalization

What's new

  • Format negative exponential numbers
  • Prevent JS exception when try to parse null with globalize.js
  • Support for case insensitive parsing of date strings

What's fixed

  • kendo.toString does not round up numbers correctly because depends on number.toFixed
  • The TypeScript return type of kendo.culture is wrong.

HierarchicalDataSource

What's fixed

  • Leaf nodes inferred from list are not marked as loaded
  • Error event does trigger when child datasource load fails

MVVM

What's new

  • Expose kendo.data.Binder to TypeScript

What's fixed

  • Event binding doesn't work in some cases with nested view models

SPA

What's fixed

  • Multiple router instances lead to duplicate callbacks

Upload

What's fixed

  • File names in Upload files list does not get encoded
  • Upload input element is not disabled on .enable(false)

Validator

What's fixed

  • Not required inputs are highlighted as such in IE8
  • Validator does not work if input has a percent symbol in the name attrbiute
  • Checkboxes is not validated correctly with jQuery 1.9.x is referenced

Web

What's fixed

  • jQuery (instead of $) is referenced in kendo.core, which breaks AMD
  • Buttons with k-state-disabled have hover and active CSS states
  • BootStrap theme is missing focus shadows
  • Retina sprites may be served improperly due to @ in name

Calendar

What's fixed

  • Navigate when setting min value equal to current and selected value is null
  • Widget disables today link if max value has different hours than zeros

ColorPicker

What's fixed

  • Inconsistent Widget resizing when font-size changes.
  • HSV picker throws "currentStyle is null or not an object" in IE 8
  • Select event is not fired when only the opacity is changed in HSV picker.

ComboBox

What's new

  • Disable auto binding if the widget cascades

What's fixed

  • Inconsistent Widget resizing when font-size changes.
  • Widget on third level does not select correct value
  • Widget does not allow to select item after filtration

DatePicker

What's fixed

  • Inconsistent Widget resizing when font-size changes.
  • Widget shows incorrectly selected date in the calendar
  • Widget disables today link if max value has different hours than zeros
  • Default date changed for all DatePickers on the page when value for one of them is selected

DateTimePicker

What's fixed

  • Inconsistent Widget resizing when font-size changes.
  • Widget shows incorrectly selected date in the calendar

DropDownList

What's new

  • Disable auto binding if the widget cascades

What's fixed

  • Inconsistent Widget resizing when font-size changes.
  • Select correct item by index when rebind widget manually
  • Widget does not apply focused state to the wrapper when in readonly state
  • Widget on third level does not select correct value
  • Widget does not select first item using MVVM source binding

Editor

What's new

  • Allow customization of default tool formatting
  • refresh() method that allows re-initialization of editing area after moving it in the DOM
  • Allow customization of foreColor/backColor palettes through tool options

What's fixed

  • Keydown event does not trigger when executing commands
  • Cannot create link from formatted text
  • Editor ViewHtml dialog scrolls the page
  • Deleting top paragraph causes next paragraph to have "line-height: 1.5;"
  • ViewHtml dialog button text is not localized
  • Fix incorrect range handling when selecting nodes at end of line in IE
  • Editor underline tool removes custom styles
  • Inserting an image which name contains a # via the ImageBrowser does not work as expected
  • Not able to remove inline styles from text in Google Chrome and Safari desktop
  • Foreground and Background color buttons are missing the unselectable attribute
  • Adding a line feed on end of line in IE9 throws JavaScript error
  • Error event is not raised when file upload fails
  • Editor selectedHtml method does not include spaces selected before text
  • Caret is out of view after creating paragraphs at end of document (IE / FF)

Grid

What's new

  • Ability to set column resize handle width
  • EditRow and editCell methods does not work if a selector is passed

What's fixed

  • Column menu should not allow hiding of all columns with field defined
  • Dropdown Popup could not be focused in grid column menu.
  • Filter menu could not be focused in navigatable grid with column menu.
  • Groups cannot be collapsed if Grid is instantiated via MVVM and bound to a DataSource instance
  • Filters with null values are send to the server
  • Manual filter of the dataSource does not add k-state-active class to the filter icon
  • Loading animation is not positioned correctly in RTL mode
  • Column resizing does not work as expected under IE when there are no records and scrolling is enabled
  • Columns cannot be resized after setting new a DataSource instance
  • Grid renders obsolete cellspacing attribute
  • Destroying the Grid does not destroy reorderable
  • Grid multi selection ctrl + click does not work on Safari (Mac)

MVVM

What's new

  • Support for binding multiselect to an array of value types.

Menu

What's fixed

  • Kendo Popup doesn't remove anchor's border-left/border-right on mouseleave
  • "k-state-highlighted" is not visible when applied to a root element

MultiSelect

What's new

  • Show initially selected values when widget uses deffered loading

What's fixed

  • MultiSelect does not trigger the blur event when the value is changed
  • Multiselect Popup opens and closes in IE8 when click on input area, but do a blur
  • MultiSelect selected items area has wrong background color
  • Binding MultiSelect to an ObservableArray of not ObservableObjects causes a JavaScript error
  • Widget clears previously selected item if value binding points to a nested property
  • Widget does not perform search if typed test is same as the previous one
  • Widget does not update correctly MVVM model if it has been filtered
  • Do not open popup on click if minLength option is not set to 0
  • Does not raise SELECT event when item is selected using keyboard
  • Scale input width when widget is initialized in hidden container

NumericTextBox

What's fixed

  • Inconsistent Widget resizing when font-size changes.
  • Support for editing negative exponential number
  • Numerictextbox spins multiple times on IE10 when arrows are clicked
  • NumericTextBox with tabindex does not tab properly on iOS devices.

What's fixed
